# Personal Expense Tracker

## Problem
Individuals often struggle to track daily expenses across multiple categories, leading to poor budgeting and overspending.

## App Concept
A web-based personal expense tracker that allows users to log expenses, categorize them, and view spending summaries.

## Must Have Features
- Add an expense
- View all expenses
- Edit an expense
- Delete an expense

## Should Have Features
- Filter expenses by category or date
- View total spending

## Could Have Features
- Monthly summaries
- Visual charts

## Target Users
- Students
- Individuals managing personal budgets
